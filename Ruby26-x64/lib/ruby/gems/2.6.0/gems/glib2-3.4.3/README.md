# Ruby/GLib2

Ruby/GLib2 is a Ruby binding of GLib-2.12.x.

## Requirements

  * Ruby >= 1.9.x: https://www.ruby-lang.org/
  * pkg-config.rb: https://github.com/rcairo/pkg-config
  * GLib >= 2.12.x: https://www.gtk.org/

## Install

```bash
gem install glib2
```
